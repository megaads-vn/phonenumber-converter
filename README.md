# Phone number converter

##Installation:
**add in file composer.json**
```
"require": {
	"megaads/phonenumber-converter": "^1.0"
}
```
##Usage:
**Register Provider**
```
#/Config/app.php
'providers' => [
    Megaads\PhonenumberConverter\PhonenumberConverterServiceProvider::class
];
```

****
```
http://domain/package/phonenumber-converter
```
